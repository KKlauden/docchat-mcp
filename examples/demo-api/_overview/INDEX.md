# JSONPlaceholder API Overview

[JSONPlaceholder](https://jsonplaceholder.typicode.com) is a free fake REST API for testing and prototyping. It provides 6 common resources, and this knowledge pack covers 3 of them.

## Available Feeds

| Feed Code | Endpoint | Description |
|-----------|----------|-------------|
| get-users | GET /users | User profiles with address and company info |
| get-posts | GET /posts | Blog posts with title and body |
| get-comments | GET /comments | Comments on posts with email and body |

## Base URL

```
https://jsonplaceholder.typicode.com
```

No authentication required. All responses are JSON.
